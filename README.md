# ホットラインAIエージェント プロトタイプ

店舗インシデントを本部へ相談するためのブラウザプロトタイプです。

## ファイル構成

- `index.html`：ブラウザで直接開けるスタンドアロン版です。
- `hotline_app.js`：スタンドアロン版の画面・分岐ロジックです。
- `hotline_ai_agent_v3.jsx`：React組み込み用のJSXコンポーネントです。

## 使い方

1. `index.html` をブラウザで開きます。
2. 大分類から順に回答します。
3. 確認画面で「この内容で本部へ送信する」を押すと、Slack通知・管理表記入のモック完了画面が表示されます。

## 実装済みの主な機能

- 7カテゴリと調剤ミス14分類の分岐
- 店番6桁バリデーション
- 服用有無、健康被害、対応状況に応じた追加質問
- 管理薬剤紛失時の患者状態質問スキップ
- 15文字以上の自由入力に対するルールベース要約確認
- 途中修正、確認画面での項目別修正
- Slack通知プレビュー、管理表24列プレビュー
- 下書きのブラウザ内自動保存
- 電話相談へのフォールバック導線

## 注意

このプロトタイプでは、Slack API・Google Sheets API・LLM APIは呼び出していません。画面上のモックとして通知済み・記入済みを表示します。
